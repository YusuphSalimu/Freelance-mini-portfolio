Freelance Portfolio – E-commerce Project
Overview

This project is a modern e-commerce frontend built to demonstrate real-world web development skills. It focuses on clean UI, interactive user experience, and scalable code structure.

Features

Dynamic product loading from JSON

Persistent shopping cart using localStorage

Add & remove cart items

Checkout flow with form validation

Responsive and modern UI design

Smooth animations and hover interactions

Clean, modular project structure

Technologies Used

HTML5

CSS3 (modern layouts & animations)

JavaScript (ES6)

JSON (data source)

How It Works

Products are loaded dynamically from a JSON file.

Cart state is saved in the browser for persistence.

Checkout simulates order processing with user feedback.

The structure allows easy backend integration.

Future Improvements

Backend integration (Node.js + Express + MongoDB)

Authentication system

Payment gateway integration
